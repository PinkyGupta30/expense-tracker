# 💰 Expense Tracker

A simple and interactive **Expense Tracker Web Application** built using **HTML, Bootstrap, and JavaScript**.
It allows users to add, edit, and delete expenses while storing the data in **Local Storage** so it persists even after refreshing the page.

---

## 🚀 Features

✔ Add new expenses
✔ Choose expense category
✔ Edit existing expenses
✔ Delete expenses
✔ Data stored in **Local Storage**
✔ Clean and responsive UI using **Bootstrap**

---

## 📂 Categories Available

Users can choose from the following categories:

* 🍔 Food
* 🎬 Entertainment
* 🏥 Health
* 🧴 Self Care

## ⚙️ How It Works

### 1️⃣ Add Expense

Enter:

* Amount
* Description
* Category

Click **Add Expense** and the expense will be saved.

---

### 2️⃣ Display Expenses

Expenses appear like this:

```
₹200 - Dinner - Food
₹500 - Movie - Entertainment
```

---

### 3️⃣ Edit Expense

Click **Edit**
The expense details will appear in the form so you can update them.

---

### 4️⃣ Delete Expense

Click **Delete** to remove the expense from:

* the list
* Local Storage
